# 🤖 FRIDAY - AI Voice Assistant

FRIDAY is a voice-controlled AI assistant built using Python, Groq API, and system automation tools.

## 🚀 Features
- Voice commands
- YouTube control
- Google search
- App launcher
- Screenshot capture
- AI conversation (Groq)

## 🛠️ Tech Stack
- Python
- Groq API
- SpeechRecognition
- Edge-TTS
- PyAutoGUI
